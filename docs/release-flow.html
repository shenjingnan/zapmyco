发布流程架构图

zapmyco · Deno 原生自动化双发布 · JSR + npm

本地发布

  • deno run -A tools/release.ts
  • 解析 conventional commits 推导版本
  • 更新 deno.json + CHANGELOG.md
  • git commit + tag + push
  • 创建 GitHub Release

CI 自动发布 (JSR)

  • 触发方式: GitHub Release published
  • deno publish 直接发布
  • OIDC 认证 (id-token: write)
  • 发布到 @zapmyco/zapmyco
  • 仅 Deno 生态

CI 自动发布 (npm)

  • 触发方式: GitHub Release published
  • dnt 转换 Deno → npm 包
  • npm publish --provenance
  • 发布到 zapmyco
  • ESM + 类型声明 + CLI bin
