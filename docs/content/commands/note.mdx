title: note — 快速笔记
description: 快速记录灵感、待办和想法的笔记系统
---

快速记录灵感、待办、想法，每条笔记是一个独立的 Markdown 文件，保存在 `~/.zapmyco/notes/` 目录下。无需 `init`，开箱即用。

```bash
# 快速记录一条笔记
zapmyco note add '修复了登录页的样式问题'

# 列出最近笔记
zapmyco note ls
```

## 子命令一览

| 子命令 | 说明 |
|--------|------|
| `add [content]` | 创建笔记，留空则使用编辑器交互输入 |
| `ls [n]` | 列出最近 n 条笔记（默认 20），`--all` 显示全部 |
| `show <id>` | 查看指定笔记的完整内容 |
| `grep <keyword>` | 搜索笔记内容 |
| `rm <id>` | 删除指定笔记 |

## 子命令详解

### `add [content]` — 创建笔记

将参数内容直接保存为笔记文件。如果不提供内容，会打开 `$EDITOR`（或 vi）进行交互式编辑。

```bash
# 快速记录
zapmyco note add '明天下午三点会议'

# 交互式编辑（打开 $EDITOR/vi）
zapmyco note add
```

**存储格式**：每条笔记是一个独立的 Markdown 文件，文件名包含创建时间。

```markdown
---
created: 2026-05-30T14:30:02+08:00
---

修复了登录页的样式问题
```

---

### `ls [n]` — 列出笔记

列出最近创建的笔记，默认显示 20 条。

```bash
# 列出最近 20 条
zapmyco note ls

# 列出最近 5 条
zapmyco note ls 5

# 显示全部笔记
zapmyco note ls --all
```

---

### `show <id>` — 查看笔记

查看指定笔记的完整内容。笔记 ID 可以从 `ls` 命令的输出中获取。

```bash
zapmyco note show 2026-05-30_143002_修复登录页样式
```

---

### `grep <keyword>` — 搜索笔记

在所有笔记中搜索关键词，返回匹配的笔记列表。

```bash
zapmyco note grep 登录
```

---

### `rm <id>` — 删除笔记

删除指定笔记。笔记 ID 可以从 `ls` 命令的输出中获取。删除后无法恢复。

```bash
zapmyco note rm 2026-05-30_143002_修复登录页样式
```
