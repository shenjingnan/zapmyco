本指南将帮助你快速安装和使用 ZapMyCo。

## 安装

ZapMyCo 提供多种安装方式，任选一种即可。

### 通过 cargo 安装

需要安装 [Rust 工具链](https://rustup.rs)：

```bash
cargo install zapmyco
```

### 一键安装脚本（推荐，无需安装 Rust）

**macOS / Linux**

```bash
curl -fsSL https://zapmyco.com/install.sh | sh
```

**Windows (PowerShell)**

在 PowerShell 中（推荐）:
```powershell
irm https://zapmyco.com/install.ps1 | iex
```

在 cmd.exe 中:
```powershell
powershell -c "irm https://zapmyco.com/install.ps1 | iex"
```

安装脚本会自动检测平台、下载对应二进制归档、验证完整性，并配置 PATH。

### 从源码编译

```bash
git clone https://github.com/shenjingnan/zapmyco.git
cd zapmyco
cargo build --release
# 产物位于 target/release/zapmyco
```

## 验证安装

安装完成后，运行以下命令验证：

```bash
zapmyco --version
# 输出示例：zapmyco 0.1.0

zapmyco --help
# 输出示例：
# 基于 Rust 的 AI 驱动命令行工具
#
# Usage: zapmyco [COMMAND]
#
# Commands:
#   greet     向指定名称打招呼
#   config    显示配置信息
#   init      初始化 LLM 配置
#   settings  显示 LLM 配置
#   run       一次性执行 AI 任务
```
