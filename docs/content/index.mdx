**ZapMyCo** 是一个基于 Rust 的 AI 驱动命令行工具，提供与 LLM 的交互式聊天体验。

## 特性

- **Rust 实现** — 单二进制文件（~5-10MB），零运行时依赖，毫秒级启动
- **AI 集成** — 内置 AI Agent 对话模式，支持流式输出
- **多平台分发** — crates.io 发布 + 5 平台预编译二进制
- **多供应商** — 内置 DeepSeek、GLM（智谱）模型，支持自定义
- **自动化发布** — GitHub Actions 驱动的自动构建、测试和发布

### 通过 cargo 安装

```bash
cargo install zapmyco
```

### 从源码编译

```bash
git clone https://github.com/shenjingnan/zapmyco.git
cd zapmyco
cargo build --release
# 产物位于 target/release/zapmyco

# 验证编译结果
./target/release/zapmyco -V

# 安装到 PATH
cargo install --path .

# 验证安装
zapmyco -V
```

## 验证安装

安装完成后，运行以下命令验证：

```bash
zapmyco --version
# 输出示例：zapmyco 0.25.1

zapmyco --help
# 输出示例：
# 基于 Rust 的 AI 驱动命令行工具
#
# Usage: zapmyco [COMMAND]
#
# Commands:
#   settings  显示 LLM 配置
#   run       一次性执行 AI 任务
#   init      初始化 LLM 配置
#   completion 生成 shell 补全脚本
#   help      显示帮助信息
```

## 快速使用

```bash
# 初始化配置
zapmyco init

# 查看帮助
zapmyco --help

# AI 对话模式
zapmyco
```
