---

Zapmyco 提供多种安装方式，你可以根据使用场景选择最适合的方式。

## 一键安装脚本（推荐，无需安装 Rust）

<Tabs items={["macOS / Linux", "Windows (PowerShell)", "Windows (cmd.exe)"]}>

<Tab>
```bash
curl -fsSL https://zapmyco.com/install.sh | sh
```
</Tab>

<Tab>
```powershell
irm https://zapmyco.com/install.ps1 | iex
```
</Tab>

<Tab>
```powershell
powershell -c "irm https://zapmyco.com/install.ps1 | iex"
```
</Tab>

</Tabs>

安装脚本会自动检测平台、下载对应二进制归档、验证完整性，并配置 PATH。

## 通过 cargo 安装

```bash
cargo install zapmyco
```

需要安装 [Rust 工具链](https://rustup.rs)。

## 从源码编译

```bash
git clone https://github.com/shenjingnan/zapmyco.git
cd zapmyco
cargo build --release
# 产物位于 target/release/zapmyco

# 验证编译结果
./target/release/zapmyco -V

# 安装到 PATH
cargo install --path .

# 验证安装
zapmyco -V
```
