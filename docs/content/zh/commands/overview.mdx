ZapMyCo 提供简洁的命令行接口，支持以下命令。

## 全局选项

| 选项 | 说明 |
|------|------|
| `--help`, `-h` | 显示帮助信息 |
| `--version`, `-V` | 显示版本号 |

## 命令速查表

| 命令 | 功能 | 需要先 init？ |
|------|------|:----------:|
| `zapmyco`（默认） | 进入交互式 AI 对话模式 | 是 |
| `zapmyco init` | 初始化 LLM 配置向导 | — |
| `zapmyco note add [content]` | 快速记录笔记（灵感/待办/想法），留空则交互式编辑 | 否 |
| `zapmyco run <content>` | 执行 AI 任务 | 是 |
| `zapmyco settings [subcommand]` | 显示或管理 LLM 配置 | 否 |
| `zapmyco config` | 显示应用配置 | 否 |
| `zapmyco upgrade` | 将 zapmyco 升级到最新版本 | 否 |
| `zapmyco uninstall` | 卸载 zapmyco | 否 |

## 使用示例

```bash
# 查看版本
zapmyco --version

# 初始化配置
zapmyco init

# 快速记录笔记
zapmyco note add '明天下午三点会议'
zapmyco note ls

# 一次性 AI 任务
zapmyco run 帮我查一下今天的日期

# 使用 Skill 系统
zapmyco run --skill code-review
zapmyco run --skill check '检查并修复代码质量问题'

# 交互式 AI 对话
zapmyco

# 查看配置
zapmyco settings
zapmyco settings path

# 查看应用配置
zapmyco config

# 升级
zapmyco upgrade

# 卸载
zapmyco uninstall

# 查看帮助
zapmyco --help
```
