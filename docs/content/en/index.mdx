**Zapmyco** is a Rust-based AI-driven command-line tool that provides an interactive chat experience with LLMs.

## Features

- **Rust Implementation** — Single binary (~5-10MB), zero runtime dependencies, millisecond startup
- **AI Integration** — Built-in AI Agent conversation mode with streaming output
- **Quick Notes** — Capture ideas, todos, and thoughts anytime, purely local Markdown storage
- **Multi-Platform Distribution** — crates.io publishing + precompiled binaries for 5 platforms
- **Multi-Provider** — Built-in DeepSeek, GLM model support, with custom provider support
- **Automated Publishing** — GitHub Actions-driven automated build, test, and release

### Build from Source

```bash
git clone https://github.com/shenjingnan/zapmyco.git
cd zapmyco
cargo build --release
# Artifact located at target/release/zapmyco

# Verify the build
./target/release/zapmyco -V

# Install to PATH
cargo install --path .

# Verify installation
zapmyco -V
```

## Verify Installation

After installation, run the following commands to verify:

```bash
zapmyco --version
# Example output: zapmyco 0.25.1

zapmyco --help
# Example output:
# A Rust-based AI-driven command-line tool
#
# Usage: zapmyco [COMMAND]
#
# Commands:
#   note      Quick note taking
#   settings  Display LLM configuration
#   run       One-shot AI task execution
#   init      Initialize LLM configuration
#   completion Generate shell completion scripts
#   help      Display help information
```

## Quick Start

```bash
# Initialize configuration
zapmyco init

# View help
zapmyco --help

# AI chat mode
zapmyco
```
