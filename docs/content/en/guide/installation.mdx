ZapMyCo offers multiple installation methods. You can choose the one that best suits your use case.

## One-Click Install Script (Recommended, No Rust Required)

<Tabs items={["macOS / Linux", "Windows (PowerShell)", "Windows (cmd.exe)"]}>

<Tab>
```bash
curl -fsSL https://zapmyco.com/install.sh | sh
```
</Tab>

<Tab>
```powershell
irm https://zapmyco.com/install.ps1 | iex
```
</Tab>

<Tab>
```powershell
powershell -c "irm https://zapmyco.com/install.ps1 | iex"
```
</Tab>

</Tabs>

The install script automatically detects the platform, downloads the corresponding binary archive, verifies integrity, and configures PATH.

## Install via cargo

```bash
cargo install zapmyco
```

Requires the [Rust toolchain](https://rustup.rs).

## Build from Source

```bash
git clone https://github.com/shenjingnan/zapmyco.git
cd zapmyco
cargo build --release
# Artifact located at target/release/zapmyco

# Verify the build result
./target/release/zapmyco -V

# Install to PATH
cargo install --path .

# Verify installation
zapmyco -V
```
