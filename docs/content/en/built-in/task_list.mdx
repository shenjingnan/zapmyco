List all current tasks and their status overview. No parameters required; directly lists all tasks and their statuses.

The task management system provides 4 tools (`task_create`, `task_get`, `task_list`, `task_update`) for breaking down complex multi-step work into trackable task units and monitoring progress in real-time.

## Output Example

```text
#1 [pending] Implement user login functionality
#2 [in_progress] Improve error handling mechanism (admin)
#3 [completed] Write unit tests [blocked by #1]
```
