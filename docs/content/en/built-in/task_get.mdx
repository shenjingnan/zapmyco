Get the details and status of a single task by ID.

The task management system provides 4 tools (`task_create`, `task_get`, `task_list`, `task_update`) for breaking down complex multi-step work into trackable task units and monitoring progress in real-time.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `task_id` | string | Yes | ID of the task to retrieve |

## Output Example

```text
Task #1: Implement user login functionality
Status: pending
Description: Implement email/password login, OAuth third-party login
Active form: Implementing login
```
