Create a new task to break down complex multi-step work into trackable task units.

The task management system provides 4 tools (`task_create`, `task_get`, `task_list`, `task_update`) for breaking down complex multi-step work into trackable task units and monitoring progress in real-time.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `subject` | string | Yes | Concise task title (e.g., "Implement user login functionality") |
| `description` | string | Yes | Detailed task description and completion criteria |
| `active_form` | string | No | Active form for progress display (e.g., "Implementing login") |

## Usage Examples

```bash
zapmyco run 'I need to create a task to track the development progress of the user login feature'
```
