Upgrade zapmyco to the latest version:

```bash
zapmyco upgrade
```

**Behavior**

Automatically fetches the latest version information from GitHub Releases, downloads and replaces the currently running binary. Download progress is displayed during the upgrade process.

**Notes**

- It is recommended to restart the terminal or reload the shell after the upgrade is complete
- If already on the latest version, the message "Already up to date" will be shown
- An internet connection is required to access the GitHub Releases API
