ZapMyCo 提供简洁的命令行接口，支持以下命令。

## 全局选项

| 选项 | 说明 |
|------|------|
| `--help`, `-h` | 显示帮助信息 |
| `--version`, `-V` | 显示版本号 |

## 命令速查表

| 命令 | 功能 | 需要先 init？ |
|------|------|:----------:|
| `zapmyco`（默认） | 进入交互式 AI 对话模式 | 是 |
| `zapmyco init` | 初始化 LLM 配置向导 | — |
| `zapmyco note add [content]` | 快速记录笔记（灵感/待办/想法），留空则交互式编辑 | 否 |
| `zapmyco run <content>` | 一次性执行 AI 任务 | 是 |
| `zapmyco settings [subcommand]` | 显示或管理 LLM 配置 | 否 |
| `zapmyco config` | 显示应用配置 | 否 |
| `zapmyco uninstall` | 卸载 zapmyco | 否 |

## 命令详解

### 交互模式（默认）

直接运行 `zapmyco`（不带任何参数）进入交互式 AI 对话模式：

```bash
zapmyco
```

启动后将进入交互式对话界面，支持多轮连续对话。输入 `/exit` 退出，`/clear` 清空对话上下文。

详细使用说明请参见 [AI 代理功能](/guide/ai-agent)。

---

### `init` — 初始化配置

交互式初始化向导，用于配置 AI 供应商和 API Key：

```bash
zapmyco init
```

向导分为三步：

1. **选择 AI 供应商** — DeepSeek / GLM（智谱）/ 自定义
2. **配置 API Key** — 支持直接输入，或使用 `${env.VAR_NAME}` 引用环境变量（推荐）
3. **选择默认模型** — 根据所选供应商过滤可用的内置模型，显示上下文窗口信息

当 `~/.zapmyco/settings.toml` 已存在时，`init` 会拒绝运行：

```
/home/user/.zapmyco/settings.toml 已存在。如需重新初始化，请先删除该文件。
```

如需重新初始化，请手动删除配置文件后再运行 `init`：

```bash
rm ~/.zapmyco/settings.toml
zapmyco init
```

向导每一步都支持 `Ctrl+C` 安全退出。

### `note` — 快速笔记

快速记录灵感、待办、想法，支持命令行一键记录和交互式编辑：

```bash
# 快速记录
zapmyco note add "修复了登录页的样式问题"

# 交互式编辑（打开 $EDITOR/vi 编辑）
zapmyco note add

# 列出最近笔记
zapmyco note ls
zapmyco note ls --all           # 显示全部
zapmyco note ls 5              # 显示最近 5 条

# 查看笔记
zapmyco note show 2026-05-30_143002_修复登录页样式

# 搜索笔记
zapmyco note grep 登录

# 删除笔记
zapmyco note rm 2026-05-30_143002_修复登录页样式
```

**存储位置**：每条笔记是一个独立的 Markdown 文件，保存在 `~/.zapmyco/notes/` 目录下，包含创建时间元信息。无需 `init`，开箱即用。

**文件格式示例**（`~/.zapmyco/notes/2026-05-30_143002_修复登录页样式.md`）：

```markdown
---
created: 2026-05-30T14:30:02+08:00
---

修复了登录页的样式问题
```

**子命令说明**

| 子命令 | 说明 |
|--------|------|
| `add [content]` | 将参数内容直接保存为笔记，留空则交互式编辑 |
| `ls [n]` | 列出最近 n 条笔记（默认 20），`--all` 显示全部 |
| `show <id>` | 查看指定笔记的完整内容 |
| `grep <keyword>` | 搜索笔记内容 |
| `rm <id>` | 删除指定笔记 |

### `run` — 一次性执行 AI 任务

一次性执行 AI 任务，完成后自动退出：

```bash
zapmyco run "用中文介绍 Rust 语言的特点"
zapmyco run --profile advanced 重构这个模块
```

**参数说明**

| 参数 | 必需 | 说明 |
|------|:--:|------|
| `<content>` | 是 | 任务描述或提示词 |
| `--profile <name>` | 否 | 指定模型配置档（对应 `settings.toml` 中 `llm.models` 的 key） |

**前置条件**：必须先通过 `zapmyco init` 完成初始化配置。

**错误场景**

- 未找到配置文件时提示运行 `zapmyco init`
- 任务描述为空时返回错误：`任务描述不能为空`

**行为说明**：AI 响应以**流式输出**方式逐字显示，完成后自动退出。

**工具支持**：`run` 命令内置了 3 个 AI Agent 工具，AI Agent 会自动判断何时使用它们：

| 工具 | 功能 |
|------|------|
| `web_fetch` | 获取网页内容并转为 Markdown |
| `web_search` | 搜索网络获取实时信息 |
| `run_command` | 在本地执行 shell 命令（需用户确认） |

详细说明请参见[内置工具](/guide/built-in-tools)。

### `settings` — 配置管理

显示或管理 LLM 运行时配置：

```bash
zapmyco settings          # 显示完整配置（API Key 自动脱敏）
zapmyco settings path     # 显示配置文件路径
zapmyco settings show     # 等同于无参数，显示完整配置
```

**子命令说明**

| 子命令 | 行为 |
|--------|------|
| 无参数 或 `show` | 读取 `~/.zapmyco/settings.toml`，API Key 自动脱敏显示（如 `sk-***`） |
| `path` | 仅输出配置文件路径 |

**脱敏规则**：API Key 显示前 3 个字符加 `***`（如 `sk-***`、`sho***`），避免敏感信息泄露。

**错误场景**

- 未知子命令会报错：`未知子命令: xxx\n可用命令: settings, settings path`
- JSON 格式错误会给出中文提示

**与 `config` 命令的区别**：
- `settings` 显示的是**用户配置**（`~/.zapmyco/settings.toml`，存储 AI 供应商和模型设置）
- `config` 显示的是**应用配置**（硬编码的调试标志、日志级别等）

---

### `config` — 应用配置

显示当前应用配置信息：

```bash
zapmyco config
# 输出示例：
# {
#   "debug": false,
#   "logLevel": "info",
#   "createdAt": "2026-05-25T..."
# }
```

**输出字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| `debug` | boolean | 调试模式开关（当前固定为 `false`） |
| `logLevel` | string | 日志级别（当前固定为 `info`） |
| `createdAt` | string | 命令执行时的 UTC 时间戳（ISO 8601 格式） |

此命令显示的是**应用内置配置**，与 `settings` 显示的用户配置不同。

---

### `uninstall` — 卸载

卸载 zapmyco，清理配置文件、安装收据和二进制文件：

```bash
zapmyco uninstall
```

卸载过程分为两个阶段：

**Phase 1 — 确认阶段**
- 询问是否保留记忆和配置（`~/.zapmyco/` 目录）
- 再次确认是否卸载
- 此阶段可按 `Ctrl+C` 安全终止，不会删除任何数据

**Phase 2 — 执行阶段**
- 删除安装收据目录 `~/.config/zapmyco/`
- 若用户选择不保留，删除 `~/.zapmyco/`（含 settings.toml、记忆等）
- 删除当前运行的二进制文件

**平台差异**

| 平台 | 二进制删除方式 |
|------|--------------|
| macOS / Linux | 自动删除自身二进制文件 |
| Windows | 提示用户手动删除二进制文件 |

## 使用示例

```bash
# 查看版本
zapmyco --version

# 初始化配置
zapmyco init

# 快速记录笔记
zapmyco note add "明天下午三点会议"
zapmyco note ls

# 一次性 AI 任务
zapmyco run 帮我查一下今天的日期

# 交互式 AI 对话
zapmyco

# 查看配置
zapmyco settings
zapmyco settings path

# 查看应用配置
zapmyco config

# 卸载
zapmyco uninstall

# 查看帮助
zapmyco --help
```
