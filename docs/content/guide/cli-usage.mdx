| 选项 | 说明 |
|------|------|
| `--help`, `-h` | 显示帮助信息 |
| `--version`, `-v`, `-V` | 显示版本号 |

## 使用方式

`zapmyco` 支持两种运行模式：

### 交互模式（默认）

直接运行 `zapmyco` 进入交互式 AI 对话模式：

```bash
zapmyco
```

启动后将进入交互式对话界面，支持多轮对话。输入 `/exit` 退出。

### 一次性执行

使用 `zapmyco run` 执行一次性任务，完成后自动退出：

```bash
zapmyco run 帮我写一个 hello world
zapmyco run --profile advanced 重构这个模块
```

### `greet <name>`

向指定名称打招呼。

```bash
zapmyco greet World
# 输出: Hello, World!

zapmyco greet 张三
# 输出: Hello, 张三!
```

### `config`

显示当前配置信息。

```bash
zapmyco config
# 输出示例：
# {
#   "debug": false,
#   "logLevel": "info",
#   "createdAt": "2026-05-25T..."
# }
```

## 使用示例

```bash
# 查看版本
zapmyco --version

# 打招呼
zapmyco greet World

# 查看配置
zapmyco config

# 交互式 AI 对话
zapmyco

# 一次性 AI 任务
zapmyco run 帮我查一下今天的日期

# 查看帮助
zapmyco --help
```
