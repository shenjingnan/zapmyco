ZapMyCo 提供简洁的命令行接口，支持以下命令。

## 全局选项

| 选项 | 说明 |
|------|------|
| `--help`, `-h` | 显示帮助信息 |
| `--version`, `-v`, `-V` | 显示版本号 |

## 命令列表

### `greet <name>`

向指定名称打招呼。

```bash
zapmyco greet World
# 输出: Hello, World!

zapmyco greet 张三
# 输出: Hello, 张三!
```

### `config`

显示当前配置信息。

```bash
zapmyco config
# 输出示例：
# {
#   "debug": false,
#   "logLevel": "info",
#   "createdAt": "2026-05-25T..."
# }
```

### `ai`

进入 AI 对话模式，与内置的 AI Agent 进行交互。

```bash
zapmyco ai
```

启动后将进入交互式对话界面，支持多轮对话。输入 `exit` 或 `quit` 退出对话模式。

详细用法请参阅 [AI 代理功能](/guide/ai-agent)。

## 使用示例

```bash
# 查看版本
zapmyco --version

# 打招呼
zapmyco greet World

# 查看配置
zapmyco config

# AI 对话
zapmyco ai

# 查看帮助
zapmyco --help
```
