ZapMyCo 提供简洁的命令行接口，支持以下命令。

## 全局选项

| 选项 | 说明 |
|------|------|
| `--help`, `-h` | 显示帮助信息 |
| `--version`, `-V` | 显示版本号 |

## 命令列表

### 交互模式（默认）

直接运行 `zapmyco` 进入交互式 AI 对话模式：

```bash
zapmyco
```

启动后将进入交互式对话界面，支持多轮对话。输入 `/exit` 退出，`/clear` 清空上下文。

### `init`

交互式初始化向导，配置 AI 供应商和 API Key：

```bash
zapmyco init
```

按提示依次选择供应商、输入 API Key、选择默认模型。

### `run <content>`

一次性执行 AI 任务，完成后自动退出：

```bash
zapmyco run 帮我写一个 hello world
zapmyco run --profile advanced 重构这个模块
```

`--profile` 参数指定模型配置档（对应 `settings.json` 中 `llm.models` 的 key）。

### `settings [subcommand]`

显示或管理 LLM 配置：

```bash
zapmyco settings          # 显示配置（API Key 自动脱敏）
zapmyco settings path     # 显示配置文件路径
```

### `greet <name>`

向指定名称打招呼。

```bash
zapmyco greet World
# 输出: Hello, World!

zapmyco greet 张三
# 输出: Hello, 张三!
```

### `config`

显示当前应用配置。

```bash
zapmyco config
# 输出示例：
# {
#   "debug": false,
#   "logLevel": "info",
#   "createdAt": "2026-05-25T..."
# }
```

## 使用示例

```bash
# 查看版本
zapmyco --version

# 初始化配置
zapmyco init

# 打招呼
zapmyco greet World

# 查看配置
zapmyco config

# 查看设置
zapmyco settings

# 交互式 AI 对话
zapmyco

# 一次性 AI 任务
zapmyco run 帮我查一下今天的日期

# 查看帮助
zapmyco --help
```
