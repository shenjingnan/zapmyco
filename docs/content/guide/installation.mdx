---

ZapMyCo 提供多种安装方式，你可以根据使用场景选择最适合的方式。

## 二进制安装（推荐）

预编译二进制文件是最便捷的安装方式，无需安装任何运行时环境。

### 一键安装脚本

```bash
curl -fsSL https://raw.githubusercontent.com/shenjingnan/zapmyco/main/install.sh | sh
```

### 指定版本或目录

```bash
# 安装指定版本
ZAPMYCO_VERSION=v0.1.0 curl -fsSL https://raw.githubusercontent.com/shenjingnan/zapmyco/main/install.sh | sh

# 安装到自定义目录
ZAPMYCO_INSTALL=~/my-tools curl -fsSL https://raw.githubusercontent.com/shenjingnan/zapmyco/main/install.sh | sh
```

### 直接下载

你也可以从 [GitHub Releases](https://github.com/shenjingnan/zapmyco/releases) 直接下载对应平台的二进制文件：

| 平台    | 架构          | 文件名 |
| ------- | ------------- | ------ |
| Linux   | x86_64        | `zapmyco-linux-x64` |
| Linux   | ARM64         | `zapmyco-linux-arm64` |
| macOS   | Apple Silicon | `zapmyco-macos-arm64` |
| macOS   | Intel         | `zapmyco-macos-x64` |
| Windows | x86_64        | `zapmyco-windows-x64.exe` |

## 通过 cargo 安装

需要安装 [Rust 工具链](https://rustup.rs)：

```bash
cargo install zapmyco
```

## 从源码编译

```bash
git clone https://github.com/shenjingnan/zapmyco.git
cd zapmyco
cargo build --release
```

编译产物位于 `target/release/zapmyco`。
