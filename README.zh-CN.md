## 介绍

本项目大部分代码由AI生成。

## 快速开始

本项目使用 Nx 进行构建和依赖管理。

### 常用命令

- 运行前端应用：`nx serve frontend`
- 构建前端应用：`nx build frontend`
- 运行测试：`nx test frontend`
- 运行受影响项目的测试：`nx affected:test`
- 查看项目依赖图：`nx graph`
