zapmyco SubAgent v3

4 action · spawn / poll(wait) / list / kill · 全并发安全 · 自动折叠输出 · ~270 行

4 个 action

  • • spawn(cli, task) → 创建子代理，立即返回 ID
  • • poll(ids[], wait_secs) → 批量查询 + 内部重试
  • • list → 全局查看所有子代理状态
  • • kill(ids[]) → SIGTERM 终止子进程
  • • 全部并发安全

poll 内部重试

  • • wait_secs=0 → 立即检查返回
  • • wait_secs=30 → 内部每 500ms 检查一次 done
  • • 超时返回部分结果（已完成的 + 仍 running 的）
  • • 省去 LLM 多轮手动 poll 的 token 消耗
  • • 多 ID 一次调用，减少工具调用次数

设计保障

  • • --subagent 标记阻断递归
  • • 进程隔离：子 Agent 崩溃不影响主 Agent
  • • list 兜底：不依赖 LLM 记忆 subagent_id
  • • kill 止损：方向错误及时终止
  • • 输出截断首行标注：LLM 第一眼知悉
